## Overview
AgriVerse is an AI-based Farmer Query Support and Advisory System developed as part of **Smart India Hackathon 2025**.  
The platform empowers farmers with real-time, multilingual, and voice-based agricultural guidance while ensuring supply chain transparency using blockchain technology.

## Problem Statement
AI Based Farmer Query Support and Advisory System  
**Problem Statement ID:** SIH25076  
**Theme:** Agriculture, Food Tech, and Rural Development  
**Category:** Software

## Solution Summary
AgriVerse provides instant AI-driven advisory to farmers on:
- Crop management
- Irrigation guidance
- Weather alerts
- Pest and disease detection (image-based)

To build trust and transparency, blockchain is used to track agricultural produce from farm to consumer. Consumers can scan QR codes to verify product origin, quality, and farming practices.
